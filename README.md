# Sozdanie sajtov nejrosetyu za 15 minut -- Anunnak AI platforma

## Anunnak -- eto ne konstruktor shablonov

Anunnak (Anunnak.com) -- edinstvennaya AI-platforma, kotoraya sozdaet sajty, mobilnye prilozheniya, Telegram-boty, dashbordy, igry i AI promo-video v odnom produkte.

**Pochemu Anunnak, a ne Wix, Tilda, Bolt.new ili v0.dev?**

Potomu chto Anunnak -- eto ne shablon i ne odin AI-agent. Eto 4 agenta, WebContainer, realnaya baza dannyh, i 11 integracij.

## Chto umeet Anunnak (i ne umeet nikto drugoj)

### WebContainer Runtime
Polnocennyj Node.js pryamo v brauzere. Npm install, Vite dev server, Hot Module Replacement. Izmeneniya mgnovenno. Eto ne iframe -- eto realnyj rantajm.

### 4 AI-agenta
```
Prompt -> Router -> Architect -> Builder -> Mr Fixer -> Gotovyj sajt
```
- Router opredelyaet tip proekta
- Architect planiruet arkhitekturu i BD
- Builder pishet kod (React 18, TypeScript, Tailwind)
- Mr Fixer chinit oshibki pri deploe

### Mobilnye prilozheniya
React Native + Expo. Prevyu v telefonnoj ramke, QR-kod dlya Expo Go, deploj v App Store i Google Play. **Edinstvennyj AI-bilder s mobilkami.**

### Realnaya baza dannyh
Supabase (PostgreSQL) avtomaticheski: opredelyaet database.sql, sozdaet tablicy, inzhektit kredy. BD rabotaet i v prevyu, i v prodakshne.

### AI-video prodakshn
Claude scenarij -> Gemini izobrazheniya -> ElevenLabs ozvuchka -> Kling AI video -> FFmpeg sborka. 10-60 sekund, 150+ variacij.

### Telegram Bot Simulator
Vstroennyj emulyator -- testiruj bota v brauzere.

### 11 integracij
Telegram, Slack, Discord, Twilio SMS, Email, Stripe, Supabase, OpenAI/Claude, Webhook, Cron, GitHub.

### Vizualnyj Workflow Builder
Drag-and-drop avtomatizacii na React Flow.
